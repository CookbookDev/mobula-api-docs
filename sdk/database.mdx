title: "Crypto Database"
icon: "database"
description: "Access to Mobula's crypto database, SQL queries, crypto prices, metadata and social data, allowing you to run SELECT queries on the data, with ORDER, LIMIT, and WHERE clauses."
---

Mobula SDK uses PostgREST standard, propulsed by the [Supabase SDK](https://supabase.com/docs/reference/javascript/select).

### Example

```js
const mobula = new Mobula("your-api-key");

mobula.db
  .from("assets")
  .select("name,symbol,price,market_cap,volume")
  .order("volume", { ascending: true })
  .limit(100)
  .then((res) => {
    console.log(res);
  });

mobula.db
  .from("assets")
  .select("investors,name,symbol")
  .match({ name: "FTT Token" })
  .then((res) => {
    console.log(res);
  });
```

### Database Schema

Only public table yet is `assets`. It contains all the assets curated by Mobula.

| Column      | Type    | Description                      |
| ----------- | ------- | -------------------------------- |
| name        | text    | Name of the asset                |
| symbol      | text    | Symbol of the asset              |
| contracts   | text[]  | List of contracts of the asset   |
| blockchains | text[]  | List of blockchains of the asset |
| decimals    | numeric | Decimals of the asset            |
