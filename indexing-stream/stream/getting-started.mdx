Mobula provides WSS support for retrieving any raw events and curated events like Swaps or Liquidity.

Coming soon.

### Endpoint details

- URL: wss://portfolio-api-wss-fgpupeioaa-uc.a.run.app
- Message (string):

```json
{
    "name": "MyWalletTransactions",
    "chainId": ["evm:1", "evm:56"],
    "raw": ["transactions", "traces", "logs" ],
    "abstract": ["swaps", "liquidity", "poolsCreated"],
};
```

    "chainId": ["evm:1", "evm:56"],
    "raw": ["transactions", "traces", "logs" ],
    "abstract": ["swaps", "liquidity", "poolsCreated"],
}`);
});

socket.addEventListener("message", (event) => {
  const data = JSON.parse(event.data);

    return;
  }
});

// No need to close the socket, it will close automatically.
