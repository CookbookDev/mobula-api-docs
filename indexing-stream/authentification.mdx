---
title: "Generate API key"
description: "To generate an API key, click on the **Create API Key** button on your [Dashboard](https://admin.mobula.fi) and assign a name to your key. This will enable you to enjoy enhanced rate limits."
---

<Steps>
  <Step title="First Step">
    Sign-in to the [Dashboard](https://admin.mobula.fi)
    <Frame>
      <img src="https://i.imgur.com/6k9DBZM.png" />
    </Frame>
  </Step>
  <Step title="Second Step">
    Create your API key
    <Frame>
      <img src="https://i.imgur.com/gtr12UF.png" />
    </Frame>
  </Step>
  <Step title="Third Step">
    Simply add your API key in the `--access-token` parameter like this `--access-token YOUR_API_KEY`

    ```cURL
    graph deploy --node http://subgraph.mobula.io:8000
    --ipfs http://subgraph.mobula.io:5001
    --access-token YOUR_API_KEY

    ```

</Step>

</Steps>
