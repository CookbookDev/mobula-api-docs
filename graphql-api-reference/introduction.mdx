You can start query the Mobula GraphQL API by using the following endpoint:

```http
https://gql.zobula.xyz/v1/graphql
```

You can find our Asset GraphQL schema [here](/graphql-api-reference/schema/assets).

Also, you can play with the API using our [interface](https://gql.zobula.xyz/).
