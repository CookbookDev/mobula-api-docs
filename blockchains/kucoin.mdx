#### KuCoin API Overview

Mobula API extends its support to KuCoin Community Chain (KCC), a decentralized public chain by the KuCoin cryptocurrency exchange. KCC offers high performance and low-cost transactions. Our API enables developers to integrate KuCoin's unique features into a wide range of blockchain applications.

#### Octopus - Market API

- **Get Market Data**: Gain real-time and historical market data for KuCoin assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple KuCoin assets in a single request.
- **Get Market Pair**: Identify trading pairs for KuCoin assets, crucial for exchange and trading platforms.
- **Get All Pairs**: List all trading pairs for KuCoin across different exchanges, offering comprehensive market insights.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in KuCoin wallets, vital for NFT platforms and digital art collectors.
- **Get Historical Net Worth**: Track the historical net worth of wallets on KuCoin.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings within KuCoin wallets.
- **Get Historical Transactions**: Access detailed transaction histories for KuCoin wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a list of all cryptocurrencies available on KuCoin.
- **Search by Name**: Easily find KuCoin assets by name, simplifying the asset discovery process.
- **Get Metadata**: Acquire detailed metadata for assets on KuCoin.

#### Example Use Cases

- **Exchange Platform Integration**: Leverage KuCoin's market data for building efficient cryptocurrency exchange platforms.
- **Wallet Functionality**: Provide comprehensive wallet services for KuCoin users, including asset management and transaction tracking.
- **DApp Development**: Integrate KuCoin's features into decentralized applications, especially those requiring high transaction throughput and low fees.
