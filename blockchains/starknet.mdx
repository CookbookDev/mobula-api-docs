### Starknet API Overview

Mobula API now offers support for Starknet, a Layer 2 scaling solution for Ethereum that leverages zk-STARK technology to enable high throughput and scalability while maintaining Ethereum's security guarantees. With Starknet, developers can build scalable and efficient decentralized applications (DApps) on Ethereum.

### Octopus - Market API

- **Access Market Data**: Retrieve real-time and historical market data for assets on Starknet, essential for applications requiring accurate financial insights and analysis.
- **Efficient Batch Data Retrieval**: Fetch market data for multiple Starknet assets in a single query, enabling comprehensive market analysis and trend identification.
- **Identify Trading Pairs**: Discover relevant trading pairs within the Starknet ecosystem, crucial for facilitating seamless transactions and trades on decentralized exchanges.
- **Comprehensive Trading Pair Listings**: Compile a comprehensive list of all trading pairs available on Starknet, providing a detailed overview of the trading environment.

### Wallet Explorer API

- **NFT Portfolio Examination**: Investigate NFT collections and assets held in Starknet wallets, important for marketplaces and collectors engaged in the NFT space.
- **Historical Wallet Valuation Tracking**: Monitor the historical valuation of wallets on Starknet, providing insights for investors and analysts interested in wallet performance.
- **Cryptocurrency Portfolio Oversight**: Manage and track cryptocurrency holdings within Starknet wallets, crucial for users managing their digital asset portfolios.
- **Detailed Transaction Histories**: Access detailed transaction histories for activities conducted on Starknet wallets, ensuring transparency and trust in financial transactions.

### Metacore - Metadata API

- **Cryptocurrency and Asset Catalog**: List all digital currencies and assets available on Starknet, aiding in asset discovery and integration for developers.
- **Asset Search by Name**: Allow for efficient asset search by name within the Starknet ecosystem, facilitating asset identification and integration.
- **Rich Metadata Retrieval**: Obtain detailed metadata for assets on Starknet, providing essential information for developers and users making informed decisions.

### Example Use Cases

- **Financial Analysis Tools**: Leverage Starknet's market data to create financial analysis tools offering insights into asset valuations, trends, and investment opportunities.
