#### Ethereum API Overview

Mobula API offers a wide array of functionalities for Ethereum, enabling developers to access vital data and perform key operations. Our endpoints are designed to provide seamless integration and efficient access to Ethereum's blockchain features.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Ethereum assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Ethereum assets in a single request.
- **Get Market Pair**: Discover trading pairs for Ethereum assets.
- **Get All Pairs**: Obtain a list of all trading pairs for Ethereum across various exchanges.
- **Get On-Chain Trades**: Track on-chain Ethereum trading activities.
- **Get Historical Market Data**: Access historical market data for detailed analysis.
- **Websocket Price Feed**: Implement real-time price feeds for Ethereum assets.

#### Wallet Explorer API

- **Get NFTs Holdings**: View Ethereum wallet NFT holdings.
- **Get Historical Net Worth**: Analyze the historical net worth of Ethereum wallets.
- **Get Crypto Holdings**: Examine Ethereum wallet cryptocurrency holdings.
- **Get Historical Transactions**: Retrieve Ethereum wallet transaction histories.
- **Websocket Wallet Feed**: Receive real-time updates on Ethereum wallet activities.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all Ethereum-based cryptocurrencies.
- **Search by Name**: Search for Ethereum assets by name.
- **Get Metadata**: Obtain detailed metadata for Ethereum assets.

#### Example Use Cases

- **Portfolio Tracking**: Manage and track Ethereum asset portfolios.
- **Market Analysis Tools**: Analyze Ethereum's market for trends and insights.
- **Wallet Functionality**: Provide Ethereum wallet services including transaction history and asset management.
- **DApp Development**: Integrate Ethereum functionalities into decentralized applications.
- **NFT Marketplaces**: Support for Ethereum-based NFT trading and management.
