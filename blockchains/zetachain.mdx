#### ZetaChain API Overview

Mobula API supports ZetaChain. Our API integration provides access to ZetaChain's distinctive features for various application needs.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for ZetaChain assets.
- **Get Market Data (batch)**: Retrieve batch market data for ZetaChain assets in one request.
- **Get Market Pair**: Discover trading pairs associated with ZetaChain assets.
- **Get All Pairs**: List all trading pairs for ZetaChain on various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in ZetaChain wallets, important for NFT marketplaces.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on ZetaChain.
- **Get Crypto Holdings**: Examine cryptocurrency holdings in ZetaChain wallets, essential for tracking and managing digital assets.
- **Get Historical Transactions**: Access detailed transaction histories for ZetaChain wallets, vital for auditing and transaction monitoring.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on ZetaChain.
- **Search by Name**: Easily search for ZetaChain assets by name, facilitating quick asset discovery.
- **Get Metadata**: Obtain detailed metadata for assets on ZetaChain, enhancing information depth for users and applications.

#### Example Use Cases

- **Stablecoin Transactions**: Utilize ZetaChain for applications requiring stablecoin transactions.
- **Market Analysis Tools**: Leverage ZetaChain's market data for comprehensive financial analysis.
- **Wallet Services**: Provide wallet functionalities specific to ZetaChain, including asset management and transaction history.
- **Decentralized Applications**: Integrate ZetaChain's unique features into DApps, especially those benefiting from stable transaction costs.
