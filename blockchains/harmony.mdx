#### Harmony API Overview

Mobula API extends its support to Harmony, known for its high throughput and low-latency blockchain platform. Our API allows developers to tap into Harmony's unique capabilities, enhancing application performance and functionality.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Harmony assets, vital for applications requiring current market information.
- **Get Market Data (batch)**: Retrieve batch data for multiple Harmony assets, perfect for applications needing a wide-angle view of the market.
- **Get Market Pair**: Discover trading pairs related to Harmony assets, crucial for exchanges and trading applications.
- **Get All Pairs**: Enumerate all trading pairs for Harmony, offering a complete perspective for market analysis tools.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in Harmony wallets, essential for NFT platforms and collectors.
- **Get Historical Net Worth**: Track the historical net worth of Harmony wallets, providing valuable financial insights.
- **Get Crypto Holdings**: Examine cryptocurrency holdings in Harmony wallets, crucial for asset management and portfolio tracking.
- **Get Historical Transactions**: Retrieve transaction histories of Harmony wallets, key for financial auditing and tracking.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on Harmony, crucial for applications providing a comprehensive market overview.
- **Search by Name**: Search for Harmony assets by name, streamlining asset discovery.
- **Get Metadata**: Acquire detailed metadata for assets on Harmony, enhancing data richness for applications.

#### Example Use Cases

- **Portfolio Management**: Manage and track portfolios with Harmony assets.
- **Market Analysis Tools**: Use Harmony's market data for comprehensive market analysis and insights.
- **Wallet Functionality**: Offer wallet services tailored to Harmony, including transaction history and asset management.
- **DApp Development**: Integrate Harmony's functionalities into decentralized applications for enhanced user experiences.
