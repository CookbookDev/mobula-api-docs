#### SmartBCH API Overview

Mobula API supports SmartBCH, a sidechain for Bitcoin Cash aiming to explore new horizons in decentralized finance. Our API facilitates integration with SmartBCH, providing access to its DeFi-oriented features.

#### Octopus - Market API

- **Get Market Data**: Access market data for SmartBCH assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple SmartBCH assets.
- **Get Market Pair**: Identify trading pairs for SmartBCH assets.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in SmartBCH wallets.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings in SmartBCH wallets.
- **Get Historical Transactions**: Retrieve transaction histories for SmartBCH wallets.

#### Example Use Cases

- **DeFi Applications**: Leverage SmartBCH for decentralized finance applications.
- **Wallet Services**: Provide wallet functionalities for SmartBCH users.
- **Market Analysis Tools**: Utilize SmartBCH data for in-depth financial analysis.
