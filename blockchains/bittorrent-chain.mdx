#### BitTorrent Chain API Overview

Mobula API offers comprehensive support for BitTorrent Chain (BTTC), a pioneering cross-chain interoperability protocol. This platform facilitates seamless interaction between major blockchains, enhancing the capabilities of decentralized applications. Our API enables easy integration with BTTC, offering developers access to its unique features and functionalities.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for BitTorrent Chain assets, critical for applications requiring current pricing and market cap information.
- **Get Market Data (batch)**: Retrieve batch market data for multiple BitTorrent Chain assets in a single request, ideal for applications needing a broad market perspective.
- **Get Market Pair**: Identify trading pairs for BitTorrent Chain assets, essential for trading applications and exchange rate calculations.
- **Get All Pairs**: Compile a list of all trading pairs for BitTorrent Chain across various exchanges, providing a comprehensive view for market analysis.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in BitTorrent Chain wallets, a must-have feature for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on BitTorrent Chain, offering valuable insights for financial tracking and analysis.
- **Get Crypto Holdings**: Examine cryptocurrency holdings within BitTorrent Chain wallets, vital for portfolio management and asset tracking.
- **Get Historical Transactions**: Access transaction histories for BitTorrent Chain wallets, crucial for auditing and financial tracking.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on BitTorrent Chain, crucial for applications that offer market overviews and asset categorization.
- **Search by Name**: Easily find BitTorrent Chain assets by name, simplifying the process of asset discovery.
- **Get Metadata**: Obtain detailed metadata for assets on BitTorrent Chain, enhancing the information richness for users and applications.
