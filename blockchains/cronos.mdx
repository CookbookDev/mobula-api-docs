#### Cronos API Overview

Mobula API offers comprehensive support for Cronos, a blockchain focused on scalability and user experience. Our API provides seamless integration with Cronos, allowing developers to access a wide array of functionalities for various applications.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Cronos assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Cronos assets.
- **Get Market Pair**: Discover trading pairs associated with Cronos assets.
- **Get All Pairs**: List all trading pairs for Cronos on various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in Cronos wallets.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Cronos.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings in Cronos wallets.
- **Get Historical Transactions**: Access the transaction history of Cronos wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on Cronos.
- **Search by Name**: Search for Cronos assets by name.
- **Get Metadata**: Obtain detailed metadata for Cronos-based assets.

#### Example Use Cases

- **Portfolio Management**: Manage and track portfolios featuring Cronos assets.
- **Market Analysis Tools**: Analyze Cronos's market for strategic insights.
- **Wallet Services**: Offer wallet functionalities specific to Cronos.
- **DApp Development**: Seamlessly integrate Cronos features into decentralized applications.

Begin by registering for a Mobula API key. Our documentation provides detailed instructions and examples for integrating Cronos functionalities.
