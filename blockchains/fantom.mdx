#### Fantom API Overview

Mobula API offers robust support for Fantom, a highly scalable and fast blockchain platform. Our API integrates seamlessly with Fantom, enabling developers to leverage its advanced features for diverse applications.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Fantom assets, critical for applications requiring up-to-date pricing and market cap information.
- **Get Market Data (batch)**: Retrieve market data for multiple Fantom assets in a single request, ideal for comprehensive market analysis.
- **Get Market Pair**: Identify trading pairs for Fantom assets, essential for trading platforms and exchange rate calculations.
- **Get All Pairs**: List all trading pairs available on Fantom across various exchanges, providing a broad view for market research tools.

#### Wallet Explorer API

- **Get NFTs Holdings**: Examine NFT holdings in Fantom wallets, pivotal for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Fantom, offering insights for financial tracking and asset valuation.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings within Fantom wallets, key for portfolio management and tracking.
- **Get Historical Transactions**: Access transaction histories for Fantom wallets, vital for auditing and transaction tracking.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a comprehensive list of cryptocurrencies on Fantom, essential for market overview applications.
- **Search by Name**: Find Fantom assets by name, facilitating asset discovery for users and applications.
- **Get Metadata**: Gather detailed metadata for Fantom-based assets, enriching the data available for users and applications.

#### Example Use Cases

- **Portfolio Management**: Utilize Fantom data for effective crypto portfolio management.
- **Market Analysis**: Leverage Fantom's market data for insightful trend analysis and predictive modeling.
- **Wallet Services**: Provide wallet functionalities for Fantom, including asset management and transaction history.
- **Decentralized Application Integration**: Seamlessly incorporate Fantom features into DApps.
