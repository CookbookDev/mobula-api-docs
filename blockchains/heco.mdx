#### HECO API Overview

Mobula API supports HECO (Huobi Eco Chain), a decentralized and cost-effective blockchain network. Our API integrates seamlessly with HECO, providing developers with a range of functionalities to explore its efficient and diverse ecosystem.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for HECO assets, essential for accurate market assessments.
- **Get Market Data (batch)**: Retrieve batch market data for multiple HECO assets, ideal for comprehensive market analysis.
- **Get Market Pair**: Discover trading pairs associated with HECO assets, crucial for trading and exchange platforms.
- **Get All Pairs**: Compile a complete list of all trading pairs on HECO across various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in HECO wallets, vital for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on HECO, providing insights for financial analysis.
- **Get Crypto Holdings**: Examine cryptocurrency holdings in HECO wallets, essential for portfolio management and tracking.
- **Get Historical Transactions**: Access transaction history for HECO wallets, important for auditing and transaction monitoring.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on HECO.
- **Search by Name**: Search for HECO assets by name, facilitating quick discovery.
- **Get Metadata**: Obtain detailed metadata for assets on HECO, enhancing user information.

#### Example Use Cases

- **DeFi Platforms**: Utilize HECO's features for developing decentralized finance applications.
- **Market Analysis Tools**: Leverage HECO's market data for financial insights and trend analysis.
- **Wallet Services**: Provide comprehensive wallet functionalities for HECO users, including asset management.
- **Trading Platforms**: Integrate HECO trading pairs into cryptocurrency exchange platforms.
