#### Aurora API Overview

Mobula API supports Aurora, an Ethereum-compatible blockchain that runs on the NEAR Protocol, combining scalability with Ethereum's robust functionality. Our API enables seamless integration with Aurora, facilitating access to a range of blockchain features for diverse applications.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Aurora assets, essential for applications requiring current pricing and market trends.
- **Get Market Data (batch)**: Retrieve batch market data for several Aurora assets in a single request, ideal for broad market analysis.
- **Get Market Pair**: Identify trading pairs for Aurora assets, crucial for trading platforms and financial applications.
- **Get All Pairs**: List all trading pairs available on Aurora across multiple exchanges, offering a comprehensive market view.

#### Wallet Explorer API

- **Get NFTs Holdings**: Explore NFT holdings in Aurora wallets, a key feature for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Aurora, providing insights into asset value changes over time.
- **Get Crypto Holdings**: View cryptocurrency holdings within Aurora wallets, essential for portfolio management and asset tracking.
- **Get Historical Transactions**: Retrieve the transaction history of Aurora wallets, vital for auditing and financial tracking.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a comprehensive list of cryptocurrencies available on Aurora.
- **Search by Name**: Search for Aurora assets by name, streamlining asset discovery.
- **Get Metadata**: Acquire detailed metadata for assets on Aurora, enhancing the richness of information for users and applications.

#### Example Use Cases

- **Decentralized Finance (DeFi) Applications**: Leverage Aurora's Ethereum compatibility for developing advanced DeFi platforms.
- **Market Analysis Tools**: Utilize Aurora's market data for in-depth financial analysis and trend prediction.
- **Wallet Functionality**: Provide comprehensive wallet services, including transaction history and asset management, for Aurora users.
- **Cross-Chain Applications**: Exploit Aurora's interoperability for applications that operate across different blockchains.
