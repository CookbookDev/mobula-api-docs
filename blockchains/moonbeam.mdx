#### Moonbeam API Overview

Mobula API supports Moonbeam, a fully Ethereum-compatible environment on Polkadot, facilitating the creation of interoperable blockchain applications. Our API provides seamless integration with Moonbeam, offering developers access to its advanced features.

#### Octopus - Market API

- **Get Market Data**: Gain access to both real-time and historical market data for Moonbeam assets.
- **Get Market Data (batch)**: Retrieve batch market data for various Moonbeam assets efficiently.
- **Get Market Pair**: Identify trading pairs for Moonbeam assets, crucial for exchanges and trading platforms.
- **Get All Pairs**: Enumerate all trading pairs for Moonbeam on different exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: Explore NFT holdings in Moonbeam wallets, important for digital art platforms and collectors.
- **Get Historical Net Worth**: Track the historical net worth of wallets on Moonbeam.
- **Get Crypto Holdings**: View cryptocurrency holdings within Moonbeam wallets.
- **Get Historical Transactions**: Access detailed transaction histories for Moonbeam wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a comprehensive list of cryptocurrencies available on Moonbeam.
- **Search by Name**: Easily find Moonbeam assets by name.
- **Get Metadata**: Acquire detailed metadata for assets on Moonbeam.

#### Example Use Cases

- **Cross-Chain DApp Integration**: Leverage Moonbeam's Ethereum compatibility for cross-chain decentralized application development.
- **Financial Analysis Tools**: Utilize Moonbeam's market data for comprehensive financial analysis.
- **NFT Marketplaces**: Support NFT trading and management on Moonbeam.
- **Wallet Functionality**: Provide enhanced wallet services for Moonbeam users, including asset tracking and transaction history.
