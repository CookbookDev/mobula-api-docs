#### Alephium API Overview

Mobula API supports Alephium. Our API integration provides access to Alephium's distinctive features for various application needs.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Alephium assets.
- **Get Market Data (batch)**: Retrieve batch market data for Alephium assets in one request.
- **Get Market Pair**: Discover trading pairs associated with Alephium assets.
- **Get All Pairs**: List all trading pairs for Alephium on various exchanges.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on Alephium.
- **Search by Name**: Easily search for Alephium assets by name, facilitating quick asset discovery.
- **Get Metadata**: Obtain detailed metadata for assets on Alephium, enhancing information depth for users and applications.

#### Getting Started

To integrate Alephium functionalities with the Mobula API, begin by registering for an API key. Our documentation provides detailed instructions and sample requests to help you get started with Alephium integration.
