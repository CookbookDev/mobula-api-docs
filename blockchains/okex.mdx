#### OKEX API Overview

Mobula API extends support to OKEX Chain, a high-performance and innovative blockchain developed by the OKEX cryptocurrency exchange. Our API offers seamless integration with OKEX Chain, allowing developers to harness its unique features for various applications, from trading systems to financial tools.

#### Octopus - Market API

- **Get Market Data**: Access up-to-date and historical market data for OKEX Chain assets, crucial for applications requiring real-time market insights.
- **Get Market Data (batch)**: Retrieve batch market data for multiple OKEX Chain assets in one query, ideal for applications that need a broad market overview.
- **Get Market Pair**: Discover trading pairs associated with OKEX Chain assets, vital for trading platforms and exchange rate calculations.
- **Get All Pairs**: List all trading pairs for OKEX Chain across various exchanges, providing a comprehensive view for market analysis.

#### Wallet Explorer API

- **Get NFTs Holdings**: Explore NFT holdings in OKEX Chain wallets, key for NFT marketplaces and digital art platforms.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on OKEX Chain, offering insights for financial tracking and valuation.
- **Get Crypto Holdings**: View cryptocurrency holdings within OKEX Chain wallets, essential for portfolio management and asset tracking.
- **Get Historical Transactions**: Access detailed transaction histories for OKEX Chain wallets, important for financial auditing and historical analysis.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a comprehensive list of cryptocurrencies available on OKEX Chain.
- **Search by Name**: Easily search for OKEX Chain assets by name, simplifying asset discovery.
- **Get Metadata**: Acquire detailed metadata for assets on OKEX Chain, enhancing information depth for users and applications.

#### Example Use Cases

- **Trading and Exchange Platforms**: Leverage OKEX Chain market data for trading insights and exchange functionalities.
- **Portfolio Management**: Utilize OKEX Chain data for managing and tracking diverse crypto portfolios.
- **Wallet Functionality**: Provide comprehensive wallet services for OKEX Chain, including transaction history and asset management.
- **Decentralized Finance (DeFi) Applications**: Seamlessly integrate OKEX Chain features into DeFi applications for enhanced user engagement.
