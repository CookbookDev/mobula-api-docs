### Solana API Overview

Mobula API proudly supports Solana, a high-performance blockchain platform known for its scalability, speed, and low transaction costs. With its innovative architecture, Solana enables developers to build decentralized applications (DApps) and projects that require fast transaction speeds and high throughput.

### Octopus - Market API

- **Real-Time Market Data Access**: Obtain real-time and historical market data for assets on Solana, crucial for applications requiring up-to-date financial insights and analysis.
- **Efficient Batch Data Retrieval**: Retrieve market data for multiple Solana assets in a single query, enabling comprehensive market analysis and trend identification.
- **Trading Pair Identification**: Identify relevant trading pairs within the Solana ecosystem, essential for facilitating seamless transactions and trades on exchanges.
- **Comprehensive Trading Pair Listings**: Compile a comprehensive list of all trading pairs available on Solana, providing a detailed overview of the trading environment.

### Wallet Explorer API

- **NFT Portfolio Examination**: Investigate NFT collections and assets held in Solana wallets, important for marketplaces and collectors engaged in the NFT space.
- **Historical Wallet Valuation Tracking**: Monitor the historical valuation of wallets on Solana, providing insights for investors and analysts interested in wallet performance.
- **Cryptocurrency Portfolio Oversight**: Manage and track cryptocurrency holdings within Solana wallets, crucial for users managing their digital asset portfolios.
- **Detailed Transaction Histories**: Access detailed transaction histories for activities conducted on Solana wallets, ensuring transparency and trust in financial transactions.

### Metacore - Metadata API

- **Cryptocurrency and Asset Catalog**: List all digital currencies and assets available on Solana, aiding in asset discovery and integration for developers.
- **Asset Search by Name**: Allow for efficient asset search by name within the Solana ecosystem, facilitating asset identification and integration.
- **Rich Metadata Retrieval**: Obtain detailed metadata for assets on Solana, providing essential information for developers and users making informed decisions.
