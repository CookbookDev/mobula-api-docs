### Secret Network API Overview

Mobula API proudly introduces support for Secret Network, a blockchain platform that prioritizes privacy and confidentiality for decentralized applications (dApps) and smart contracts. With its unique privacy features, Secret Network enables developers to build privacy-preserving applications across various industries.

### Octopus - Market API

- **Access Market Data**: Retrieve real-time and historical market data for assets on Secret Network, essential for applications requiring accurate financial insights and analysis.
- **Efficient Batch Data Retrieval**: Fetch market data for multiple Secret Network assets in a single query, enabling comprehensive market analysis and trend identification.
- **Identify Trading Pairs**: Discover relevant trading pairs within the Secret Network ecosystem, crucial for facilitating seamless transactions and trades on decentralized exchanges.
- **Comprehensive Trading Pair Listings**: Compile a comprehensive list of all trading pairs available on Secret Network, providing a detailed overview of the trading environment.

### Wallet Explorer API

- **NFT Portfolio Examination**: Investigate NFT collections and assets held in Secret Network wallets, important for marketplaces and collectors engaged in the NFT space.
- **Historical Wallet Valuation Tracking**: Monitor the historical valuation of wallets on Secret Network, providing insights for investors and analysts interested in wallet performance.
- **Cryptocurrency Portfolio Oversight**: Manage and track cryptocurrency holdings within Secret Network wallets, crucial for users managing their digital asset portfolios.
- **Detailed Transaction Histories**: Access detailed transaction histories for activities conducted on Secret Network wallets, ensuring transparency and trust in financial transactions.

### Metacore - Metadata API

- **Cryptocurrency and Asset Catalog**: List all digital currencies and assets available on Secret Network, aiding in asset discovery and integration for developers.
- **Asset Search by Name**: Allow for efficient asset search by name within the Secret Network ecosystem, facilitating asset identification and integration.
- **Rich Metadata Retrieval**: Obtain detailed metadata for assets on Secret Network, providing essential information for developers and users making informed decisions.
