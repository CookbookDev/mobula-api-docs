#### Avalanche API Overview

Mobula API extends comprehensive support to the Avalanche blockchain, providing a suite of tools for developers to effectively utilize its robust and high-speed functionalities. Our API ensures seamless integration with Avalanche's unique features, catering to a broad spectrum of data and operational needs.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Avalanche assets. Ideal for applications needing up-to-the-minute pricing and market cap information.
- **Get Market Data (batch)**: Retrieve market data for multiple Avalanche assets in a single request, enhancing efficiency for applications requiring a comprehensive market overview.
- **Get Market Pair**: Identify trading pairs for Avalanche assets, crucial for trading applications and exchanges.
- **Get All Pairs**: Enumerate all trading pairs for Avalanche across various exchanges, providing a holistic view for market analysis tools.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings within Avalanche wallets, a key feature for NFT platforms and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Avalanche, offering valuable insights for financial tracking and analysis applications.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings in Avalanche wallets, essential for portfolio management and tracking.
- **Get Historical Transactions**: Access transaction histories for Avalanche wallets, fundamental for auditing and financial tracking applications.
- **Websocket Wallet Feed**: Receive real-time updates on Avalanche wallet activities, ideal for applications requiring instant wallet monitoring.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all Avalanche-based cryptocurrencies, vital for applications providing a market overview and asset analysis.
- **Search by Name**: Locate Avalanche assets by name, simplifying asset discovery for users and applications.
- **Get Metadata**: Acquire detailed metadata for Avalanche-based assets, critical for applications that display enriched data and detailed asset information.

#### Example Use Cases

- **Portfolio Management**: Manage and track portfolios featuring Avalanche assets.
- **Market Analysis Tools**: Employ Avalanche market data for insightful trend analysis and market predictions.
- **Wallet Services**: Provide wallet functionalities specific to Avalanche, including asset management and transaction history.
- **DApp Integration**: Incorporate Avalanche's features into decentralized applications for enhanced user experiences.
