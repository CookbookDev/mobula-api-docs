#### Celo API Overview

Mobula API provides robust support for Celo, a blockchain ecosystem focused on increasing cryptocurrency adoption among smartphone users. Our API facilitates seamless integration with Celo, enabling access to a range of features tailored for mobile-based financial applications.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for Celo assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Celo assets in one request.
- **Get Market Pair**: Discover trading pairs associated with Celo assets.
- **Get All Pairs**: List all trading pairs available on Celo across various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in Celo wallets, perfect for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Celo.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings in Celo wallets.
- **Get Historical Transactions**: Access detailed transaction histories for Celo wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on Celo.
- **Search by Name**: Search for Celo assets by name.
- **Get Metadata**: Obtain detailed metadata for Celo-based assets.

#### Example Use Cases

- **Mobile Finance Applications**: Utilize Celo's mobile-centric features for developing financial apps.
- **Portfolio Management**: Manage and track portfolios featuring Celo assets.
- **Market Analysis Tools**: Leverage Celo's market data for strategic financial insights.
- **Wallet Functionality**: Offer comprehensive wallet services for Celo users.
