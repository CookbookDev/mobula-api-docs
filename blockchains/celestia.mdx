### Celestia API Overview

Mobula API now extends its support to Celestia, a cutting-edge blockchain platform focused on enabling decentralized finance (DeFi) solutions and applications. With Celestia, developers gain access to robust infrastructure and tools for building innovative financial products and services.

### Octopus - Market API

- **Real-Time Market Data Access**: Obtain real-time and historical market data for assets on Celestia, crucial for applications requiring accurate financial insights and analysis.
- **Efficient Batch Data Retrieval**: Fetch market data for multiple Celestia assets in a single query, enabling comprehensive market analysis and trend identification.
- **Identify Trading Pairs**: Discover relevant trading pairs within the Celestia ecosystem, crucial for facilitating seamless transactions and trades on decentralized exchanges.
- **Comprehensive Trading Pair Listings**: Compile a comprehensive list of all trading pairs available on Celestia, providing a detailed overview of the trading environment.

### Wallet Explorer API

- **NFT Portfolio Examination**: Investigate NFT collections and assets held in Celestia wallets, important for marketplaces and collectors engaged in the NFT space.
- **Historical Wallet Valuation Tracking**: Monitor the historical valuation of wallets on Celestia, providing insights for investors and analysts interested in wallet performance.
- **Cryptocurrency Portfolio Oversight**: Manage and track cryptocurrency holdings within Celestia wallets, crucial for users managing their digital asset portfolios.
- **Detailed Transaction Histories**: Access detailed transaction histories for activities conducted on Celestia wallets, ensuring transparency and trust in financial transactions.

### Metacore - Metadata API

- **Cryptocurrency and Asset Catalog**: List all digital currencies and assets available on Celestia, aiding in asset discovery and integration for developers.
- **Asset Search by Name**: Allow for efficient asset search by name within the Celestia ecosystem, facilitating asset identification and integration.
- **Rich Metadata Retrieval**: Obtain detailed metadata for assets on Celestia, providing essential information for developers and users making informed decisions.
