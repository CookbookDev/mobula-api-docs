#### Velas API Overview

Mobula API extends its support to Velas, a high-performance and scalable blockchain platform that leverages artificial intelligence in its consensus mechanism. Our API enables developers to integrate Velas's innovative features into a wide range of applications, from DApps to financial systems.

#### Octopus - Market API

- **Get Market Data**: Gain access to real-time and historical market data for Velas assets.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Velas assets efficiently.
- **Get Market Pair**: Discover trading pairs associated with Velas assets, crucial for exchanges and trading applications.
- **Get All Pairs**: List all trading pairs for Velas across various exchanges, offering comprehensive market insights.

#### Wallet Explorer API

- **Get NFTs Holdings**: Explore NFT holdings in Velas wallets, important for NFT platforms and digital art collectors.
- **Get Historical Net Worth**: Track the historical net worth of wallets on Velas.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings within Velas wallets.
- **Get Historical Transactions**: Access detailed transaction histories for Velas wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a list of all cryptocurrencies available on Velas.
- **Search by Name**: Find Velas assets by name, streamlining the asset discovery process.
- **Get Metadata**: Acquire detailed metadata for assets on Velas, enhancing the richness of information for users.
