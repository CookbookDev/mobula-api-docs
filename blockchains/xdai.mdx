#### xDai API Overview

Mobula API supports xDai, a unique blockchain that offers stable transactions with its native stablecoin. Our API integration provides access to xDai's distinctive features for various application needs.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for xDai assets.
- **Get Market Data (batch)**: Retrieve batch market data for xDai assets in one request.
- **Get Market Pair**: Discover trading pairs associated with xDai assets.
- **Get All Pairs**: List all trading pairs for xDai on various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in xDai wallets, important for NFT marketplaces.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on xDai.
- **Get Crypto Holdings**: Examine cryptocurrency holdings in xDai wallets, essential for tracking and managing digital assets.
- **Get Historical Transactions**: Access detailed transaction histories for xDai wallets, vital for auditing and transaction monitoring.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on xDai.
- **Search by Name**: Easily search for xDai assets by name, facilitating quick asset discovery.
- **Get Metadata**: Obtain detailed metadata for assets on xDai, enhancing information depth for users and applications.

#### Example Use Cases

- **Stablecoin Transactions**: Utilize xDai for applications requiring stablecoin transactions.
- **Market Analysis Tools**: Leverage xDai's market data for comprehensive financial analysis.
- **Wallet Services**: Provide wallet functionalities specific to xDai, including asset management and transaction history.
- **Decentralized Applications**: Integrate xDai's unique features into DApps, especially those benefiting from stable transaction costs.

#### Getting Started

To integrate xDai functionalities with the Mobula API, begin by registering for an API key. Our documentation provides detailed instructions and sample requests to help you get started with xDai integration.
