#### Arbitrum API Overview

Mobula API offers comprehensive support for Arbitrum, a Layer 2 scaling solution designed to improve the scalability and efficiency of Ethereum. Our API facilitates seamless integration with Arbitrum, enabling developers to leverage its enhanced transaction throughput and reduced gas costs for Ethereum-based applications.

#### Octopus - Market API

- **Get Market Data**: Access real-time and historical market data for assets on Arbitrum, crucial for applications requiring up-to-date pricing and market trends.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Arbitrum assets in a single request, ideal for platforms that need a comprehensive market overview.
- **Get Market Pair**: Identify trading pairs for Arbitrum assets, essential for trading applications and exchange platforms.
- **Get All Pairs**: Compile a list of all trading pairs for Arbitrum across various exchanges, providing a broad market perspective.

#### Wallet Explorer API

- **Get NFTs Holdings**: View NFT holdings in Arbitrum wallets, a must-have feature for NFT marketplaces and digital art platforms.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Arbitrum, offering valuable insights for financial tracking and asset valuation.
- **Get Crypto Holdings**: Examine cryptocurrency holdings within Arbitrum wallets, vital for portfolio management and asset tracking.
- **Get Historical Transactions**: Access detailed transaction histories for Arbitrum wallets, important for auditing and financial analysis.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: List all cryptocurrencies available on Arbitrum, crucial for applications offering market overviews and asset categorization.
- **Search by Name**: Easily search for Arbitrum assets by name, simplifying the asset discovery process.
- **Get Metadata**: Obtain detailed metadata for assets on Arbitrum, enhancing the richness of information for users and applications.
