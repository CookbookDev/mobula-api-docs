#### Mantle API Overview

Mobula API provides robust support for Mantle, a blockchain platform renowned for its high performance and scalability. Our API facilitates seamless integration with Mantle, enabling developers to harness its capabilities for a wide array of applications, from enterprise solutions to decentralized apps.

#### Octopus - Market API

- **Get Market Data**: Gain access to real-time and historical market data for Mantle assets, essential for applications that need accurate and up-to-date market insights.
- **Get Market Data (batch)**: Retrieve batch market data for multiple Mantle assets efficiently, ideal for platforms requiring a comprehensive market overview.
- **Get Market Pair**: Discover trading pairs associated with Mantle assets, crucial for trading platforms and financial applications.
- **Get All Pairs**: List all trading pairs available on Mantle across various exchanges.

#### Wallet Explorer API

- **Get NFTs Holdings**: Explore NFT holdings in Mantle wallets, vital for NFT marketplaces and collectors.
- **Get Historical Net Worth**: Analyze the historical net worth of wallets on Mantle.
- **Get Crypto Holdings**: Monitor cryptocurrency holdings within Mantle wallets.
- **Get Historical Transactions**: Access detailed transaction histories for Mantle wallets.

#### Metacore - Metadata API

- **Get All Cryptocurrencies**: Obtain a list of all cryptocurrencies available on Mantle.
- **Search by Name**: Find Mantle assets by name, streamlining the asset discovery process.
- **Get Metadata**: Acquire detailed metadata for assets on Mantle.

#### Example Use Cases

- **Enterprise Blockchain Solutions**: Leverage Mantle for developing scalable and efficient enterprise-level blockchain applications.
- **Market Analysis Tools**: Utilize Mantle's market data for comprehensive financial analysis and trend identification.
- **Decentralized App Development**: Integrate Mantle's features into DApps, taking advantage of its high scalability.
- **Wallet Services**: Provide enhanced wallet functionalities for Mantle users, including asset management and transaction tracking.
