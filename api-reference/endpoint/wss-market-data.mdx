<Tip> This endpoint is only available to Growth and Enterprise plans. </Tip>

### Endpoint details

- URL: wss://general-api-wss-fgpupeioaa-uc.a.run.app
- Message (string):

```json
{
  "type": "market",
  "authorization": "YOUR-API-KEY",
  "payload": {
    "assets": [
      { "name": "Bitcoin" },
      {
        "address": "0x059fd118aa8988f5e97b66fcd489765111e069c9",
        "blockchain": "1"
      }
    ],
    "interval": 15
  }
}
```

Parameters:

- `assets`: array of assets to track - can be queried by name, symbol, or address + blockchain
- `interval`: interval in seconds between updates (min 5s, default 30s)

### Data model

Data reponse is the same as [market/multi-data](/api-reference/endpoint/market-multi-data)
