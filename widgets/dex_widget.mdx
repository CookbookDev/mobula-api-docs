<Steps>
  <Step title="First Step">
  
    Copy the following code
    
    ```
    <iframe
      scrolling="no"
      height="381px"
      width="100%"
      src="https://mobula.fi/iframe/swap?bgColor=white&bgTitle=lightgray&bgBox=whitesmoke&borderColor=darkgray&fontMain=black&fontSecondary=gray&bgButton=lightgray"
      frameBorder="0"
    ></iframe>
    ```

  </Step>
  <Step title="Second Step">
  
    You can customize the widget to make it fit your needs, to do so, you can
    change the following parameters:

    - `bgMain`: background color of the widgets

    - `bgBox`: background color of the box

    - `bgTitle`: background color of the title container

    - `bgButton`: background color of the button

    - `fontMain`: font color of the main text

    - `fontSecondary`: font color of the secondary text

    - `borderColor`: border color of the widgets

  </Step>
</Steps>

<Tip>
  You can also let the Mobula colors by default by removing the parameters you
  don't want to change
</Tip>
